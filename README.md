# Template Undangan Adat Indonesia (HTML/CSS)

Koleksi template undangan bertema adat Indonesia yang dibuat dengan HTML dan CSS. Setiap file merupakan desain berbeda yang menonjolkan nuansa budaya dari berbagai daerah.

## Isi Proyek

- Balinese Tropical Invitation.html
- Betawi Festive Invitation.html
- Javanese Batik Invitation.html
- Toraja Weaving Invitation.html

## Cara Pakai

1. Buka salah satu file HTML di browser.
2. Sesuaikan teks (nama, tanggal, lokasi, dll.) langsung di file HTML.
3. Simpan perubahan, lalu refresh browser.

## Struktur Sederhana

Setiap file HTML berisi:
- Struktur undangan lengkap (judul, nama, waktu, lokasi).
- Styling CSS di dalam file yang sama (inline `<style>`).

## Catatan

Template ini bersifat statis (tanpa backend). Cocok untuk kebutuhan undangan sederhana, demo, atau inspirasi desain.

## Lisensi

Silakan gunakan dan modifikasi sesuai kebutuhan.
